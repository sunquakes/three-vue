## 文档 

访问 [three-vue.sunquakes.com](https://three-vue.sunquakes.com).

## 安装

### 安装 `Three.js`

```bash
pnpm install three
```

### 安装 `three-vue`

```bash
pnpm install three-vue
```

## 快速开始

- 在`Vue 3`中使用.

```vue
<template>
  <div class="scene">
    <scene></scene>
  </div>
</template>

<script lang="ts" setup></script>

<style>
.scene {
  margin-top: 10px;
  width: 100%;
  height: 300px;
}
```
