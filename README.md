## Documentation

Visit [three-vue.sunquakes.com](https://three-vue.sunquakes.com).

## Install

### Install `Three.js`

```bash
pnpm install three
```

### Install `three-vue`

```bash
pnpm install three-vue
```

## Getting Started

- Use it in `Vue 3`.

```vue
<template>
  <div class="scene">
    <scene></scene>
  </div>
</template>

<script lang="ts" setup></script>

<style>
.scene {
  margin-top: 10px;
  width: 100%;
  height: 300px;
}
```
